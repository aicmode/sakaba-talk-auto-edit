# 「あー、二階堂で」→「二階堂で」修正

- 対象: 出力 9.97〜10.32 秒に残っていた「あー」(素材11「じゃあ二階堂で」の
  「じゃあ」の尻尾。クリップが「じゃあ」の途中=素材 5.968 秒から始まっていたため)
- 「二階堂で」の実発話開始(「に」の頭)= 出力 10.322〜10.327 秒
  (素材11とのxcorr照合 offset+4.002s、whisper開始スイープ、ピッチ検定 140Hz定常母音
  →10.34以降別パターン、の3手法で確定)
- 処理(scripts/remove_nikaido_aa.py):
  - 9.960〜10.233 秒: デジタル無音(直前 9.84〜9.97 秒の無音と同じ扱い)
  - 10.233 秒(シーン7カット)〜10.322 秒: 素材11のルームトーン(4.95秒〜、
    レベル照合 gain x1.00)を10msフェードインで充当
  - 右端15msの等パワークロスフェードで「二階堂で」へ接続。「に」の頭は無傷
- 「二階堂で」はテロップ表示(10.233秒〜)の約0.09秒後に開始。見本の
  発話開始(約10.0〜10.24秒)と同等の自然なタイミング
- 映像は stream copy(新旧の映像ストリームMD5一致=無変更)。モノクロ演出・
  商品画像・ズーム・テロップ・他シーンの音声はすべて不変
- 検証: 全ストリームデコードエラーなし、2450フレーム・81.67秒維持、
  10.322秒以降の音声は旧版と一致(-37dBのAAC再エンコードノイズのみ)、
  パッチ外の音声diffは-45dBFS以下
- 出力: `output/prototype.mp4`(検証後に旧版を置換・削除済み)
