## 削除した重複動作／提供シーンの変更前後

旧版は見本に存在しない `39.433 / 40.200 / 41.333 / 43.033` 秒の細切れ境界を追加し、
素材8と素材14の別アングルに含まれる同じ提供動作を連続させていた。このため、グラスが
店員側へ戻り、「渡す・受け取る・マットへ置く」が再実行されたように見えていた。

修正版は次の一方向の動作にした。

1. 39.433〜40.533: 素材6で混ぜ終わる（提供動作を先行表示しない）
2. 40.533〜41.700: 素材14で店員が一度だけ差し出す
3. 41.700〜42.867: 素材8の寄りでグラスをマットへ一度だけ置く
4. 42.867〜46.933: 素材8で受取後の客が礼を言い、一口飲む

削除した旧区間は、素材8@4.500〜5.267、素材14@3.533〜4.666、
素材8@16.263〜19.501を組み合わせた重複提供／動作ジャンプである。

## 実再生・目視確認

- 39.4〜74.4秒を、見本／出力の左右並列映像として0.2秒間隔（5fps）で全フレーム確認。
- 39.4〜43.1秒: 混ぜる→差し出す→一度だけ置く→客、の順でグラス位置の逆戻りなし。
- 43〜70秒: 見本と同じ話者ショット、口の動き、テロップ切替を確認。
- 特に48〜57秒の質問／返答、60〜64.5秒の礼／乾杯／飲む、65.9〜70.1秒の退店会話を通し確認。
- 黒画面「お会計中」、その後の挨拶も維持。

## 見本との差が残る箇所

- 見本固有の一部デジタルズーム／クロップ、短いディゾルブ、カラー調整は元素材の
  フルフレーム再構築では完全一致しない。動作内容、話者、順序、音声、テロップ時刻を優先した。
- 0〜39.433秒は今回の保護対象なので、見本側との差があっても変更していない。
- 後半音声は欠落防止を最優先し、見本の完成ミックス（BGMを含む）を使用。39.433秒の接続点のみ
  前半の復元音声から見本マスターへ切り替わる。
